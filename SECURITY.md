# Security Policy

## Supported Versions

Currently, only version **1.0.1** is supported. All security updates are released for this version.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.1   | :white_check_mark: |
| < 1.0.1 | :x:                |

## Reporting a Vulnerability

If you discover a vulnerability in the library:

1. **Do not publish** the details publicly until a fix is released.
2. **Report it to us** via our Discord: **[https://discord.gg/6TfNC2Ng8V](https://discord.gg/6TfNC2Ng8V)**
   - Create a message in the appropriate security channel or contact a moderator directly
3. **What you'll get**:
   - Acknowledgment within **48 hours**
   - Status updates every **5 days**
   - A fix release (target: **14 days** from acknowledgment)
   - Credit in release notes (you can request anonymity)
4. **If declined**: We'll explain why (not a vulnerability, already fixed, or low severity).

We appreciate all vulnerability reports and commit to respecting the reporter's privacy.
